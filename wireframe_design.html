Wireframe & Design

Wireframe :-

The wireframe for this project was created by me using my own creative thinking to structure the layout and user flow. It focuses on clear navigation, content hierarchy, and a seamless user experience.

Final UI Design :-

The final design reflects India's 26th Republic Day spirit, incorporating tricolor elements (saffron, white, and green), engaging visuals, and a modern layout. It includes a hero section, event details, and interactive elements to enhance user engagement.

This is a self-practice project, and both the wireframe and design were created entirely by me using my own ideas and creative thinking.
